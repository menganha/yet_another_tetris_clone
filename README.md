Tetris clone made with C++ and the SDL 2.0 library.

Here's a nice demo!

![demo](./demo.gif)

## Requirements

  * A GNU/Linux Distribution
  * The gcc C++ compiler
  * SDL2 Library

## Build instructions

  * Just run `make`
